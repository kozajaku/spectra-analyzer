Spectra analyzing tool

Welcome to the Home page of Spectra analyzing tool. With this tool you can download spectra from virtual observatory archives using astronomical protocols SSAP and DataLink. You can visualise downloaded spectra and interactively inspect the possibility of their dimensionality reduction using method of continual wavelet transformation.

The tool currently supports following format of astronomical spectra for inspection:

  • .vot - VOTABLE tabledata encoded
  • .vot - VOTABLE binary encoded
  • .fit - image/fits format
  • .fits - application/fits format
  • .csv - comma separated values
  • .txt - tab separated values
  • .asc - two spaces separated values
The tool matches specific reader by matching spectrum file extensions. Please ensure that your spectrum file for inspection has correctly set its extension.
